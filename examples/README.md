# Examples

Runnable examples demonstrating VecTrade Python SDK features.

## Setup

```bash
pip install vectrade
export VECTRADE_API_KEY="vq_live_..."
```

## Examples

| File | Description |
|------|-------------|
| [quickstart.py](quickstart.py) | Fetch a quote and display price info |
| [async_portfolio.py](async_portfolio.py) | Concurrent async requests for multiple symbols |
| [streaming_ai.py](streaming_ai.py) | Stream AI analysis in real-time |
| [options_chain.py](options_chain.py) | Options chain with high-volume filtering |
| [screener.py](screener.py) | Stock screener for undervalued tech |
| [developer_usage.py](developer_usage.py) | Monitor API usage and manage keys |

## Running

```bash
python examples/quickstart.py
python examples/async_portfolio.py
```
